Functionality:- The app opens with a homescreen having three game levels, beginner, intermediate and advanced. On clicking, each of them takees to a new game view having different sets of mines according to the game difficulty level. Each gameview has a timer and reset button. Reset button resets the gird, resetting the emine positions randomly again, and the timer clocks the total time a user takes to complete the game. There is an alertbox added to the backkey button which will prevent the user from accidentally quitting the game.
